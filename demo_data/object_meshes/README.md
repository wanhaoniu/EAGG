# Demo Object Meshes

These seven `.stl` meshes match the demo point clouds. They are used
only for visualization, so quickstart renders show object surfaces rather than
only sparse point clouds.

- `003_cracker_box.stl`
- `004_sugar_box.stl`
- `005_tomato_soup_can.stl`
- `006_mustard_bottle.stl`
- `011_banana.stl`
- `024_bowl.stl`
- `025_mug.stl`
